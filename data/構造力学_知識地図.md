# 建築専門1 構造力学 — 知識地図（Knowledge Map）

> 対象：2013年～2026年 全11問の過去問分析
> 目的：試験範囲の把握と知識ネットワークの構築
> ※ 解説・解法の詳細は省略し、試験戦略に使える構造に焦点を当てる

---

# 1. 知識ポイント一覧

## Ⅰ. 断面の性質
- 断面2次モーメント（I）の計算（長方形、円、正三角形、中空断面）
- 弾性断面係数（Ze）
- 塑性断面係数（Zp）
- 断面係数（Z = I / y_max）
- 中立軸の位置決定
- 図心（重心）の位置

## Ⅱ. 応力とひずみ

- 曲げ応力度 σ = My / I
- せん断応力度 τ = VQ / It（または τ = V / A 直接）
- ヤング係数 E と剛性 EA・EI
- ポアソン比
- ひずみ ε = σ / E（フックの法則）
- 線膨張係数 α と熱ひずみ ε = αΔT

## Ⅲ. 静力学の基礎

- 力の平衡条件（ΣFx=0, ΣFy=0, ΣM=0）
- 支点反力の計算（ピン、ローラー、固定）
- 力の合成と分解

## Ⅳ. 梁の曲げ

- 曲げモーメント M(x) の算出
- せん断力 V(x) の算出
- 曲げモーメント図（M図）の描画
- せん断力図（Q図）の描画
- 単純梁・片持ち梁・両端固定梁の応力図
- 等分布荷重・集中荷重による曲げ

## Ⅴ. 変形（たわみ）

- 撓み（たわみ）の計算
- 鉛直変位 δ の計算
- 水平変位の計算
- ひずみエネルギー法（虚功の原理）
- 部材角 θ の計算
- バネ支持における変形

## Ⅵ. 不静定構造

- 不静定次数の判定
- たわみ角法（ slope-deflection method ）
- 剛比 k の計算
- 固定端モーメント（FEM）
- 未知数の数と平衡条件の関係
- 軸剛性 EI/L² と曲げ剛性 EI の使い分け

## Ⅶ. トラス構造

- トラスの名称（プラット、ワーレン、キングポスト、フィンク、フィーレンデール等）
- 節点法による軸力計算
- 切断法による軸力計算
- 零杆の判定
- 軸力の正負（引張＋、圧縮ー）
- 反力計算（ピン・ローラー支持）

## Ⅷ. 座屈

- オイラー座屈荷重 Pcr = π²EI / L²
- 座屈長さ L の決定（支持条件による有条件による有効長さ係数）
- 座屈応力度 σcr = Pcr / A

## Ⅸ. 熱応力と熱変形

- 線膨張による自由伸縮 ΔL = αLΔT
- 両端固定時の熱応力 σ = EαΔT
- バネ支持時の熱変位と熱応力
- 構造フレームへの熱膨張の影響

## Ⅹ. 複合・統合問題

- 二種材料の複合梁（接着面、ヤング係数の違い、降伏応力度の違い）
- 軸力と曲げモーメントの同時作用
- 塑性中立軸と弾性中立軸の違い
- 完全弾塑性体の解析

# 2. 各知識ポイントの解説

### 断面2次モーメント（I）

**【それは何か】**
断面が曲げに対してどれだけ抵抗できるかを表す幾何学的性質。断面形状だけで決まる。

**【なぜそうなるのか】**
I = ∫y²dA で定義される。中立軸から遠い部分の面積ほど曲げに大きく寄与する。

**【試験での出題傾向】**
- 基本形状（長方形 bh³/12、円 πd⁴/64）の暗記が必須
- 正三角形の計算も出る（2013年）
- 中空断面は外側マイナス内側で求める

**【関連する知識】**
→ 断面係数 Z、曲げ応力度 σ、撓みの計算に直接使用
→ 塑性断面係数 Zp と対比して学ぶ

---

### 曲げモーメント（M）とせん断力（V）

**【それは何か】**
梁内部に生じる内力。Mは曲げの強さ、Vは横方向の剪断力。

**【なぜそうなるのか】**
外力に対して梁内部にモーメントとせん断力が平衡として生じる。Mの変化率がV（dM/dx = V）。

**【試験での出題傾向】**
- M図・Q図の描画はほぼ毎年出る
- 特に単純梁・片持ち梁の基本パターンを完璧に
- 不静定構造のM図も頻出（2016、2017、2018、2019）

**【関連する知識】**
→ 曲げ応力度 σ = My/I の計算に使う
→ たわみ角法でもモーメント計算が中心
→ 変形図の描画にもつながる

---

### 支点反力の計算

**【それは何か】**
構造物を支える支座に生じる反力。静定構造では3つの平衡方程式で解ける。

**【なぜそうなるのか】**
構造物が静止しているため、外力と反力の合力・合力がゼロになる。

**【試験での出題傾向】**
- ほぼ全問で最初の一歩として出る
- ピン支持（水平・鉛直2成分）、ローラー（鉛直1成分）、固定（M+水平+鉛直3成分）の区別

**【関連する知識】**
→ M図・Q図描画の前提
→ トラスの反力計算にも同様に適用

---

### たわみ角法

**【それは何か】**
不静定骨組のモーメントをたわみ角θと回転角φを使って求める方法。

**【なぜそうなるのか】**
部材端のたわみ角とモーメントの関係を式で表し、節点のモーメント平衡を解くことで不静定力を求める。

**【試験での出題傾向】**
- 2019年に直接出題
- 剛比k、固定端モーメントMf、標準剛度K₀の理解が必須
- 節点のモーメント平衡 ΣM = 0 が鍵

**【関連する知識】**
→ 不静定構造の解析手法
→ 剚比 k = (EI/L) / K₀ で計算
→ 固定端モーメントは荷重条件から求める

---

### トラス構造

**【それは何か】**
ピン接合された部材からなる構造。部材には軸力（引張・圧縮）のみが作用する。

**【なぜそうなるのか】**
ピン接合のため曲げモーメントが生じず、各部材は軸方向の力のみで平衡を保つ。

**【試験での出題傾向】**
- 2022年・2026年に大問として出題
- 2026年はトラスの名称も問われる（知識問題）
- 軸力計算、反力計算、座屈の組み合わせ問題が頻出

**【関連する知識】**
→ 節点法・切断法で軸力を求める
→ オイラー座屈と組み合わせて座屈荷重も出る
→ 2026年は節点変位（バネ定数的なアプローチ）も出た

---

### オイラー座屈

**【それは何か】**
細長い柱（圧縮材）が急に横に変形する現象。

**【なぜそうなるのか】**
圧縮荷重がある臨界値に達すると、微小な横変位が拡大して不安定になる。

**【試験での出題傾向】**
- 2022年に直接出題（座屈長さと座屈荷重）
- トラスの圧縮材の座屈判定
- 支持条件による有条件による有効長さ係数の理解が必須

**【関連する知識】**
→ Pcr = π²EI / L²（オイラーの公式）
→ トラスの軸力計算と組み合わせ
→ 断面2次モーメント I と直結

---

### 熱応力と熱変形

**【それは何か】**
温度変化により材料が膨張・収縮しようとするとき、拘束があると応力が生じる。

**【なぜそうなるのか】**
自由に膨張できる場合と異なり、両端が固定されていると熱膨張を阻止されるため応力が発生。

**【試験での出題傾向】**
- 2015年・2025年に大問として出題
- 2025年はフレーム構造への熱膨張の影響を問う総合問題
- α（線膨張係数）の値を知識として問うこともある（コンクリート 1×10⁻⁵/℃）

**【関連する知識】**
→ ε = αΔT（自由熱ひずみ）
→ 両端固定時の熱応力 σ = EαΔT
→ バネ支持・フレームでの応用

---

### 塑性解析

**【それは何か】**
材料が降伏した後の断面の挙動。弾性範囲を越えた時の耐力計算。

**【なぜそうなるのか】**
応力度が降伏応力度に達した部分から塑性化が進み、断面全体が塑性化すると崩壊する。

**【試験での出題傾向】**
- 2014年に複合梁での塑性断面係数・塑性中立軸を問う問題
- 完全弾塑性体の仮定
- 弾性中立軸と塑性中立軸の違い

**【関連する知識】**
→ 塑性断面係数 Zp は断面全体が降伏したときの曲げ耐力
→ 弾性断面係数 Ze との比 Zp/Ze が形状係数
→ 軸力と曲げの同時作用（P-M相関）

# 5. 勉強戦略の提案

## 優先順位（時間がない場合の攻略順）

1. **支点反力 + M図・Q図描画**（全問の基盤）
2. **変位計算**（虚功の原理をマスター）
3. **基本断面の断面2次モーメント**（暗記で対応）
4. **トラスの軸力計算**（2022・2026で大問化）
5. **たわみ角法**（不静定構造に対応）
6. **熱応力**（2015・2025で大問化、αの値も暗記）

## 暗記すべき公式

| 公式 | 用途 |
|------|------|
| I = bh³/12（長方形） | 断面2次モーメント |
| I = πd⁴/64（円） | 断面2次モーメント |
| σ = My / I | 曲げ応力度 |
| Pcr = π²EI / L² | オイラー座屈 |
| ΔL = αLΔT | 熱膨張 |
| σ = EαΔT | 両端固定時の熱応力 |
| α ≈ 1×10⁻⁵ /℃（コンクリート） | 線膨張係数 |
