## Part I. 断面の性質

### 断面2次モーメント（I）

**【それは何か】**
断面が曲げに対してどれだけ抵抗できるかを表す幾何学的性質。断面形状だけで決まる。

**【なぜそうなるのか】**
I = ∫y²dA で定義される。中立軸から遠い部分の面積ほど曲げに大きく寄与する。

**【試験での出題傾向】**
- 基本形状（長方形 bh³/12、円 πd⁴/64）の暗記が必須
- 正三角形の計算も出る（2013年）
- 中空断面は外側マイナス内側で求める

**【関連する知識】**
→ 断面係数 Z、曲げ応力度 σ、撓みの計算に直接使用
→ 塑性断面係数 Zp と対比して学ぶ

**【体系的カバー】**
- 断面1次モーメントと図心の計算
- 断面の主軸・主軸座標変換
- 断面係数（Z = I / y_max）・塑性断面係数（Zp）
- 弾性断面係数 Ze と形状係数 Zp/Ze

### 応力度とひずみ度

**【それは何か】**
部材内部に生じる単位面積あたりの力（応力）と、変形の割合（ひずみ）の関係。

**【核心概念】**
- 曲げ応力度 σ = My / I
- せん断応力度 τ = VQ / It（または τ = V / A 直接）
- ヤング係数 E と剛性 EA・EI
- ポアソン比 ν
- フックの法則 ε = σ / E
- 線膨張係数 α と熱ひずみ ε = αΔT

**【試験での出題傾向】**
- 応力度計算は2015年・2024年に頻出
- 主応力（モーるの応力円）の計算も重要

**【体系的カバー】**
- 軸方向応力度 σ = P/A
- 曲げ応力度 σ = My/I
- せん断応力度 τ = VQ/It
- 主応力とモーるの応力円
- 曲げと軸力の組み合わせ応力

## Part II. 静力学の基礎

### 力の基本 🌟

**【それは何か】**
力の合成・分解・釣り合い条件を用いて、構造物に作用する外力と反力を求める。

**【核心概念】**
- 力の平衡条件（ΣFx=0, ΣFy=0, ΣM=0）
- 力のモーメント — 任意点周りのモーメント計算は全解析の基盤
- 3ヒンジ構造物の反力 — 静定構造物の反力計算で頻出

**【試験での出題傾向】**
- ほぼ全問で最初の一歩として出る
- ピン支持（水平・鉛直2成分）、ローラー（鉛直1成分）、固定（M+水平+鉛直3成分）の区別

**【関連する知識】**
→ M図・Q図描画の前提
→ トラスの反力計算にも同様に適用

### 支点反力の計算

**【それは何か】**
構造物を支える支座に生じる反力。静定構造では3つの平衡方程式で解ける。

**【試験での出題傾向】**
- 全問の出発点。計算の第一歩として必須
- 支点条件の判定、構造物の静定/不静定の判定、自由度の計算

## Part III. 梁の曲げ — 静定梁

### せん断力と曲げモーメント 🌟🌟

**【それは何か】**
梁内部に生じる内力。Mは曲げの強さ、Vは横方向の剪断力。

**【なぜそうなるのか】**
外力に対して梁内部にモーメントとせん断力が平衡として生じる。dV/dx = -w(x), dM/dx = V(x) の微分関係が成立。

**【試験での出題傾向】**
- M図・Q図の描画はほぼ毎年出る（6回以上出現）
- 特に単純梁・片持ち梁の基本パターンを完璧に
- 不静定構造のM図も頻出（2016、2017、2018、2019）

**【関連する知識】**
→ 曲げ応力度 σ = My/I の計算に使う
→ たわみ角法でもモーメント計算が中心
→ 変形図の描画にもつながる

**【体系的カバー】**
- 片持梁：集中荷重・等分布荷重・変分布荷重・モーメント荷重
- 単純梁：集中荷重・等分布荷重・変分布荷重・モーメント荷重
- 荷重・せん断力・曲げモーメントの微分関係によるSFD/BMD描画

### 静定ラーメン 🌟🌟

**【それは何か】**
梁と柱が剛接合されたフレーム構造のうち、静定条件を満たすものの内力解析。

**【核心概念】**
- ゲルバー梁（Hinge Beam）— 中間ヒンジを持つ梁のモーメント図の描き方
- 3ヒンジラーメン — 反力未知数=釣り合い方程式数で静定となる典型例
- 片持梁系・単純梁系の静定ラーメン

**【試験での出題傾向】**
- ゲルバー梁のヒンジ位置でのモーメント=0条件の利用
- 合成ラーメンの解析

## Part IV. トラス構造

### トラス構造 🌟🌟

**【それは何か】**
ピン接合された部材からなる構造。部材には軸力（引張・圧縮）のみが作用する。

**【なぜそうなるのか】**
ピン接合のため曲げモーメントが生じず、各部材は軸方向の力のみで平衡を保つ。

**【試験での出題傾向】**
- 2022年・2026年に大問として出題
- 2026年はトラスの名称も問われる（知識問題）
- 軸力計算、反力計算、座屈の組み合わせ問題が頻出

**【関連する知識】**
→ 節点法・切断法で軸力を求める
→ オイラー座屈と組み合わせて座屈荷重も出る
→ 2026年は節点変位（バネ定数的なアプローチ）も出た

**【体系的カバー】**
- トラスの種類（プラット・ワーレン・キングポスト・フィンク・フィーレンデール等）
- 節点法（Method of Joints）による軸力計算
- 截面法（Method of Sections）による軸力計算
- 零応力部材の判定
- 軸力の正負（引張＋、圧縮ー）

## Part V. 部材の変形

### 変形（たわみ） 🌟

**【それは何か】**
モーメント・せん断力・軸力による部材の変形量を計算し、変形の定性的挙動を理解する。

**【試験での出題傾向】**

**【関連する知識】**

→ 曲率と変形の関係 κ = M/EI
→ モールの定理で変形図から変位を幾何学的に求める

**【体系的カバー】**
- 撓み（たわみ）の計算
- 鉛直変位 δ の計算
- 水平変位の計算
- ひずみエネルギー法（虚功の原理）
- 部材角 θ の計算
- バネ支持における変形
- 曲率と変形の関係
- モールの定理

## Part VI. 不静定構造 — 理論と解析

### 仕事とエネルギー 🌟🌟

**【それは何か】**
力と変位の関係を「仕事（エネルギー）」の観点から捉え、不静定構造の解析手法の理論的基盤を構築する。

**【核心概念】**
- 外力仕事と内力仕事（ひずみエネルギー）— W_ext = W_int の関係は仮想仕事の原理の根幹
- 力学的エネルギー保存則 — 弾性範囲内での力と変位の関係を保証する基本法則

**【試験での出題傾向】**
- 外力仕事の計算（荷重-変形グラフの面積）
- ひずみエネルギーの計算
- エネルギー保存則の適用

### 仮想仕事の原理（単位外力法）🌟🌟🌟

**【それは何か】**
構造物の任意の変位を求めるための最も汎用的な手法。仮想的な単位外力を作用させ、実際の内力分布との組み合わせで変位を算出する。

**【核心概念】**
- 単位外力法（Unit Load Method）— δ = Σ(N̄·N·L)/(EA) + Σ(M̄·M·dx)/(EI)
- 不静定構造の力法（Flexibility Method）の理論的根幹
- 相反定理（Maxwell's Reciprocal Theorem）— 構造解析の簡素化に寄与

**【試験での出題傾向】**
- 単位外力法による特定点の変位計算
- トラス構造の変位計算
- 仮想仕事の原理を用いた不静定次数の判定
- 力法（Force Method）の基本方程式を導く過程で必ず出題

### 不静定構造の基本 🌟🌟

**【それは何か】**
静定条件だけでは解けない構造（反力数 > 釣り合い方程式数）を「変形の適合条件」を追加して解く。

**【核心概念】**
- 変形の適合条件による解法（力法/Flexibility Method）— 余剰力（不静定力）を未知数とし、変形の連続条件から求める
- 不静定次数の判定
- 未知数の数と平衡条件の関係

**【試験での出題傾向】**
- 不静定次数の判定
- 単純な不静定梁の余剰力求め
- 連続梁の解析

### たわみ角法（Slope Deflection Method）🌟🌟🌟

**【それは何か】**
不静定骨組のモーメントをたわみ角θと回転角φを使って求める方法。

**【なぜそうなるのか】**
部材端のたわみ角とモーメントの関係を式で表し、節点のモーメント平衡を解くことで不静定力を求める。

**【核心概念】**
- 基本式 — M_AB = (2EI/L)(2θ_A + θ_B - 3ψ) + FEM_AB
- 節点方程式 — 各節点で ΣM = 0 を立てる
- 連層ラーメンの解法 — 側移を伴う不静定ラーメンの定石
- 剛比 k = (EI/L) / K₀ の計算
- 固定端モーメント FEM の設定

**【試験での出題傾向】**
- 2019年に直接出題
- 剛比k、固定端モーメントMf、標準剛度K₀の理解が必須
- 節点のモーメント平衡 ΣM = 0 が鍵

**【関連する知識】**
→ 不静定構造の解析手法
→ 剚比 k = (EI/L) / K₀ で計算
→ 固定端モーメントは荷重条件から求める
→ 軸剛性 EI/L² と曲げ剛性 EI の使い分け

### 固定モーメント法（Moment Distribution Method / Hardy Cross法）🌟🌟🌟

**【それは何か】**
たわみ角法の連立方程式を「反復計算」で解く簡便法。各節点でモーメントを隣接部材に「分配」し、「伝達」を繰り返して収束させる。

**【核心概念】**
- 分配係数（Distribution Factor）— 節点に作用するモーメントを各部材に分配する比率
- 伝達係数（Carry-over Factor）— 一端に加えたモーメントが他端に伝達される比率（固定端=1/2、ピン端=0）
- 対称な連続梁の簡略化
- 他端ピン部材の扱い

**【試験での出題傾向】**
- 連続梁のモーメント分布表の作成
- 分配係数・伝達係数の計算
- ラーメンの固定モーメント法による解法

### D値法（Portal Method の発展）🌟🌟

**【それは何か】**
水平力（地震力・風力）が作用するラーメン構造において、柱・梁の剛性比に基づき各層の水平力を柱に分配する実用的近似法。

**【核心概念】**
- D値（剛性値）— 各柱の水平抵抗能を示す値。梁の剛性との比で決定
- 反曲点高さ（y値）— 柱の曲げモーメント図において、モーメントが零となる高さ
- 一様なラーメンの柱のD値
- 一般のラーメン（壁あり/柱剛性不同）のD値
- 耐震壁のD値

**【試験での出題傾向】**
- 水平力が作用するラーメンの柱・梁モーメントの計算

## Part VII. 熱応力と熱変形

### 熱応力と熱変形

**【それは何か】**
温度変化により材料が膨張・収縮しようとするとき、拘束があると応力が生じる。

**【なぜそうなるのか】**
自由に膨張できる場合と異なり、両端が固定されていると熱膨張を阻止されるため応力が発生。

**【試験での出題傾向】**
- 2015年・2025年に大問として出題
- 2025年はフレーム構造への熱膨張の影響を問う総合問題
- α（線膨張係数）の値を知識として問うこともある（コンクリート 1×10⁻⁵/℃）

**【関連する知識】**
→ ε = αΔT（自由熱ひずみ）
→ 両端固定時の熱応力 σ = EαΔT
→ バネ支持・フレームでの応用

**【体系的カバー】**
- 線膨張による自由伸縮 ΔL = αLΔT
- 両端固定時の熱応力 σ = EαΔT
- バネ支持時の熱変位と熱応力
- 構造フレームへの熱膨張の影響

## Part VIII. マトリクス法

### 剛性マトリクス法 🌟🌟🌟

**【それは何か】**
構造物を「節点変位」の未知数とし、各部材の剛性マトリクスを組み上げて全体剛性方程式 [K]{u} = {F} を解く数値解析手法。コンピュータ構造解析の基礎。

**【核心概念】**
- 部材の剛性マトリクス — トラス部材（軸力のみ）と梁部材（せん断+曲げ）のマトリクス導出
- 全体剛性マトリクスの組立 — 境界条件（支点条件）の適用
- 任意方向を向く部材 — 座標変換（回転マトリクス）を用いた一般化

**【試験での出題傾向】**
- バネの剛性マトリクスの導出
- 平面トラスの全体剛性マトリクスの組立
- せん断力と曲げを受ける部材の剛性マトリクス
- 境界条件の適用と節点変位の計算

## Part IX. 終局強度

### 断面の終局強度 🌟🌟

**【それは何か】**
弾性解析を超え、材料が塑性領域に達した場合の断面の最大耐力（終局モーメント・終局軸力）を求める。

**【核心概念】**
- 応力度-ひずみ度関係の非線形化 — 鉄骨・RC断面での塑性化の進行
- 軸力と曲げモーメントを受ける部材の終局強度 — N-M相関曲線（P-M Interaction Diagram）

**【試験での出題傾向】**
- 2014年に複合梁での塑性断面係数・塑性中立軸を問う問題
- 完全弾塑性体の仮定
- 弾性中立軸と塑性中立軸の違い

**【関連する知識】**
→ 塑性断面係数 Zp は断面全体が降伏したときの曲げ耐力
→ 弾性断面係数 Ze との比 Zp/Ze が形状係数
→ 軸力と曲げの同時作用（P-M相関）

### 構造物の終局強度（崩壊メカニズム）🌟🌟

**【それは何か】**
構造物全体の崩壊荷重を求める。不静定構造では塑性ヒンジの発生により静定化が進み、最終的に崩壊に至るメカニズムを解析する。

**【核心概念】**
- 塑性ヒンジ（Plastic Hinge）— 断面が終局モーメントに達すると塑性ヒンジが発生
- 崩壊メカニズム — 最小の塑性ヒンジ数で崩壊に至るメカニズムを仮定し、虚Work原理で崩壊荷重を求める

**【試験での出題傾向】**
- 単純梁・片持梁の崩壊荷重
- 1次不静定構造の崩壊荷重
- 連続梁の崩壊メカニズムの判定
- ラーメンの水平崩壊荷重

## Part X. 座屈

### オイラー座屈 🌟🌟🌟

**【それは何か】**
細長い柱（圧縮材）が急に横に変形する現象。

**【なぜそうなるのか】**
圧縮荷重がある臨界値に達すると、微小な横変位が拡大して不安定になる。

**【核心概念】**
- オイラー座屈荷重 P_cr = π²EI / (κL)²
- 端部条件係数 κ による座屈荷重の変動
- 非弾性座屈 — 実際の材料では応力度が比例限を超えた場合の座屈荷重の低下

**【試験での出題傾向】**
- 2022年に直接出題（座屈長さと座屈荷重）
- トラスの圧縮材の座屈判定
- 支持条件による有条件による有効長さ係数の理解が必須

**【関連する知識】**
→ Pcr = π²EI / L²（オイラーの公式）
→ トラスの軸力計算と組み合わせ
→ 断面2次モーメント I と直結

## Part XV. 勉強戦略の提案

### 学習順序の推奨

```
【Phase 1: 基盤】力の基本 → 部材の内力（梁・トラス） → 断面の応力 → 部材の変形
        ↓
【Phase 2: 静定構造】静定梁のSFD/BMD → 静定トラス → 静定ラーメン
        ↓
【Phase 3: 不静定の理論】仕事とエネルギー → 仮想仕事の原理（単位外力法）
        ↓
【Phase 4: 不静定の解析】たわみ角法 → 固定モーメント法 → D値法
        ↓
【Phase 5: 数値解析】剛性マトリクス法（マトリクス法）
        ↓
【Phase 6: 終局と設計】終局強度 → 崩壊メカニズム → 座屈 → 設計応用
```

### 優先順位（時間がない場合の攻略順）

1. **支点反力 + M図・Q図描画**（全問の基盤）
2. **変位計算**（虚功の原理をマスター）
3. **基本断面の断面2次モーメント**（暗記で対応）
4. **トラスの軸力計算**（2022・2026で大問化）
5. **たわみ角法**（不静定構造に対応）
6. **熱応力**（2015・2025で大問化、αの値も暗記）
