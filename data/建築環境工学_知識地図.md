## 1️⃣ 建築伝熱

### 基本知識

#### 1-1 熱伝導の基礎
- **熱伝導率（λ）** [W/(m·K)]
  - 空気: ~0.025 W/(m·K)
  - コンクリート: ~1.5 W/(m·K)
  - ガラス: ~0.8 W/(m·K)
  - ウレタンフォーム: ~0.025 W/(m·K)

- **熱伝導の基本式**
  ```
  q = λ × ΔT / d
  q: 熱流束 [W/m²]
  ΔT: 温度差 [K]
  d: 壁厚 [m]
  ```

#### 1-2 熱伝達と総合熱伝達率
- **熱伝達**: 壁面と空気の間で対流と放射によって行われる伝熱
- **総合熱伝達率（h）** [W/(m²·K)]
  - 内表面: hi ≈ 9 W/(m²·K)
  - 外表面: he ≈ 23 W/(m²·K)

#### 1-3 熱貫流率（U値）
- **定義**: U = 1 / (1/hi + Σ(d/λ) + 1/he)
- **単位**: W/(m²·K)
- **多層壁の計算**: 熱抵抗の直列合成

#### 1-4 壁体表面温度の計算
```
Ts = Ti - q/hi = To + q/he
```

## 2️⃣ 換気と通風

### 基本知識

#### 2-1 必要換気量
```
Q = N / (Cin - Cout)
Q: 必要換気量 [m³/hまたはL/h]
N: 汚染物質発生量
Cin: 室内許容濃度
Cout: 室外濃度
```

#### 2-2 CO2濃度の計算
- **定常状態**: C = Cout + G/Q
- **非定常状態**: C(t) = Cout + (C0-Cout)e^(-Qt/V) + G/Q(1-e^(-Qt/V))

#### 2-3 3種の機械換気方式
| 方式 | 給気 | 排気 | 特徴 |
|------|------|------|------|
| 第1種 | 機械 | 機械 | 熱交換が可能、気密性が高い |
| 第2種 | 機械 | 自然 | 室内正圧、汚染空気の流入防止 |
| 第3種 | 自然 | 機械 | 台所、浴室で主に使用 |

#### 2-4 自然換気
- **風力換気量**: Q = Cd × A × √(2ΔP/ρ)
- **温度差換気（煙突効果）**: Q = Cd × A × √(2gH(Ti-To)/Ti)

#### 2-5 風圧係数
- 風上側: Cp ≈ +0.6
- 風下側: Cp ≈ -0.4

## 3️⃣ 採光・照明

### 基本知識

#### 3-1 測光量の体系
| 用語 | 記号 | 単位 | 定義 |
|------|------|------|------|
| 光束 | F | lm | 光源から放出される全光束 |
| 照度 | E | lx | 受照面の単位面積あたりの光束 |
| 光度 | I | cd | 点光源のある方向の発光強度 |
| 輝度 | L | cd/m² | 面光源の単位面積あたりの発光強度 |
| 光束発散度 | R | lm/m² | 面光源の単位面積あたりの光束 |

#### 3-2 照度計算
- **点光源**: E = I × cosθ / r²
- **均等拡散面**: E = πL

#### 3-3 昼光率（Daylight Factor）
```
D = (Ed / Eo) × 100%
Ed: 室内照度
Eo: 全天空照度
```

#### 3-4 照明設計計算
```
E = (F × n × U × M) / A
E: 平均照度 [lx]
F: 照明器具の光束 [lm]
n: 照明器具の台数
U: 照明率
M: 保守率
A: 床面積 [m²]
```

#### 3-5 室指数
```
K = (W × H) / (H × (W + L))
W: 室幅, L: 室長, H: 照明器具から作業面までの高さ
```

## 4️⃣ 建築音響

### 基本知識

#### 4-1 音の基本量
- **音圧レベル**: Lp = 20 log₁₀(p/p₀) [dB]
- **音響パワーレベル**: Lw = 10 log₁₀(W/W₀) [dB]
- **基準値**: p₀ = 2×10⁻⁵ Pa, W₀ = 10⁻¹² W

#### 4-2 残響時間
- **セービンの式**: T₆₀ = 0.161V / A
- **アイリングの式**: T₆₀ = 0.161V / (-S×ln(1-ā))
- **定義**: 音源停止後、音圧レベルが60dB低下するまでの時間

#### 4-3 吸音率
```
α = (Ei - Er) / Ei = Et + Ea / Ei
α: 吸音率 (0~1)
Ei: 入射エネルギー
Er: 反射エネルギー
```

#### 4-4 音響透過損失（遮音）
- **質量則**: R = 20 log₁₀(f×m) - 43 [dB]
- **f**: 周波数 [Hz]
- **m**: 面密度 [kg/m²]

#### 4-5 等価吸音面積
```
A = Σ(αi × Si)
A: 等価吸音面積 [m²]
αi: 各面の吸音率
Si: 各面の面積
```

## 5️⃣ 日射・日照

### 基本知識

#### 5-1 太陽位置の計算
- **太陽高度角**: sin h = sinφ sinδ + cosφ cosδ cosω
- **方位角**: cos A = (sin h sinφ - sinδ) / (cos h cosφ)

#### 5-2 相当外気温度
```
T_eq = T_o + (α × I - R) / he
T_eq: 相当外気温度
T_o: 外気温度
α: 日射吸収率
I: 日射量
R: 夜間放射量
he: 外表面熱伝達率
```

#### 5-3 日射受熱量
- **各方位面**: 直達日射、天空日射、反射日射を考慮
- **南向き鉛直面**: 冬至日 > 夏至日（北半球）

#### 5-4 日射遮蔽係数
- **SC値**: 3mm透明ガラスを基準
- **値が大きいほど遮蔽効果が低い**

## 6️⃣ 湿気・結露

### 基本知識

#### 6-1 湿度関連用語
| 用語 | 定義 |
|------|------|
| 絶対湿度 | 単位体積の湿潤空気中の水蒸気質量 |
| 相対湿度 | 実際の水蒸気圧/飽和水蒸気圧 ×100% |
| 露点温度 | 水蒸気が凝結し始める温度 |
| 湿球温度 | 断熱飽和温度 |

#### 6-2 結露の判断
- **表面結露**: 壁面温度 < 露点温度
- **内部結露**: 壁体内部の水蒸気圧 > 飽和水蒸気圧

#### 6-3 結露防止設計
- **防湿層の位置**: 断熱材の室内側（冬季）
- **二重窓**: 外側に気密性の高いサッシュ

#### 6-4 空気線図（h-x図）
- 湿潤空気の状態解析に使用
- 露点温度、エンタルピーなどを照査可能

## 7️⃣ 快適条件

### 基本知識

#### 7-1 温熱環境の6要素
| 側面 | 要素 |
|------|------|
| 環境側 | ①気温 ②湿度 ③気流 ④放射温度 |
| 人体側 | ⑤代謝量 ⑥着衣量 |

#### 7-2 代謝率の単位
- **met**: 1 met = 58.2 W/m²（成人男子の座位安静時）
- **clo**: 1 clo = 0.155 m²·K/W（服装の熱抵抗）

#### 7-3 快適性指標
- **作用温度（OT）**: 気温と放射温度を統合した等価温度
- **PMV（予測平均温冷感）**: 6要素を考慮した総合指標
- **等価温度**: 相対湿度100%、風速0m/sの仮想的な環境の室温

#### 7-4 人体の熱収支
```
M - W = C + R + E + S
M: 代謝産熱
W: 外部仕事
C: 対流散热
R: 放射散热
E: 蒸発散热
S: 熱蓄積
```
