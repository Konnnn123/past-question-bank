# 問題 1 Question 1

近年，特に地方都市で郊外型の大型店舗の出店が目覚ましい。このような大型店舗を設計する際には，駐車場の十分な確保が必須である。適正な駐車場規模(台数)を算出するに当り，どのようなデータを収集し、また、どのように計算を行うべきだろうか。

(a) 以下のデータリストから, 特に有効と思われるデータを選び, それらを用いた計算式を示しなさい.

(b) 上記で算出した駐車場規模 (台数) を用いる際に, 留意しなければならない事項を述べなさい.

Recently, lot of large suburban shopping malls are crowded with customers in Japan. When designing such shopping malls, it is important to secure a sufficient size of car parking space. To identify the appropriate car parking size (the number of slots), answer the following questions, both (a) and (b). 

(a) Select the most suitable data from the following data list, and explain how to calculate the appropriate size from such data. 

(b) Discuss the points of attention when using the above calculation method. 

<データリスト / data list>

敷地面積 / total site area

容積率 / floor-area ratio

店舗の総売り場面積 / total store space of the mall

店舗の出店テナント数 / number of store in the mall

来店客の平均滞在時間 / average time spent in the mall

年間来店客数 / annual number of customers

店舗周辺 5km 圈内の総世帯数 / total number of households within 5km distance from the mall

店舗周辺 5km 圈内の競合店舗数 / total number of competing stores within 5km distance from the mall

最寄駅からの距離 / distance from the nearest station

ピーク時の1時間当り自動車到着台数 / number of arrival cars at peak hour
