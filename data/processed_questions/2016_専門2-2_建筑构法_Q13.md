# 問題 8 Question 8

鉄筋コンクリート構造住宅の浴室の二種類の構法について問う。

在来構法とは、浴槽、浴室防水材料、浴室仕上げ材料、設備部品、器具.

配管、開口部品等の構成部品を現場に搬入し、加工し、組み合わせて浴室を構成していく構法である。

工業化構法：バスユニット等の構成部品を現場に搬入し、浴室を構成していく構法である。

在来構法、工業化構法それぞれについて、現場での作業手順を、工程の順番に作業を列挙する形で説明せよ(並行する作業は適宜順番を設定してよい)。各作業は如何なる職種の技能技術者（例、大工、左官）が担当するのかについても、想像できるる範囲で記すこと。

Question on building methods of bath room in reinforced concrete structural houses. 

Conventional method: one-site assemble of raw material level building elements such as bath-tub, water-roof materials, finishing materials, equipment, appliances, pipes, doors and sashes 

Industrialized method: On-site installation of prefabricated bath-room module. 

Explain site-assembly/ site installation process by listing a set of activities by sequential order (In case more than two activities can be done concurrently, you can assume the order flexibly). If possible, explain respectively what kind of site specialist (such as carpenter, plasterer, etc.) is appropriate for each activity.
